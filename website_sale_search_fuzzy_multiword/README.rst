This module allows to use fuzzy search functionality with multi-word keywords
This fuzzy search, like the original one, is to be applied for website searches
That also means that it is not possible to separate different keywords with a whitespace

Usage
=====

Just install this module and you will be able to use multi-word keywords in the website searchbar
