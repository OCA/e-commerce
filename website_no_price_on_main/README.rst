========================
No Price on main website
========================

* Identify a website as "main", all prices will not be shown on this website.
* The "main" websites will have another flow for carts, other than review > shipping & billing > payment 
  but will be review > Request Quotation > confirmations.
* all prices will be hidden , in carts, suggested products, lists.
* non-main websites will mantain the normal flow.


Usage
=====
Install module and set one (or more) websites as main website.
