=================================
Website Sale - Quantity Parameter
=================================

This module adds support for a ``quantity`` URL parameter to all website shop
product pages. When the parameter's value is a valid number, the quantity input
on the page will immediately reflect this value.

Usage
=====

To see the module in action:

1. Go to any website shop product page
2. Append the ``quantity`` parameter to the URL, making sure to provide a
   valid value
3. Hit ``Enter`` to load the page using the new URL
4. Notice that the quantity input reflects the value of the parameter
