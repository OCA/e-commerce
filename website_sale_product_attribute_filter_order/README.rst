Configuration
=============

#. Go to the eCommerce (/shop)
#. Drop down the 'Customize' menu
#. Enable the 'Product Attribute's Filters Marked First' option

Usage
=====

#. Go to the eCommerce
#. Set one or more attributes on
#. You can see that all enabled attribute values are bubbled up to the first position in the list.

  .. figure:: https://raw.githubusercontent.com/OCA/e-commerce/15.0/website_sale_product_attribute_filter_order/static/description/attributes00.png
