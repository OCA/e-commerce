====================================
Webshop Product Filter Extension
====================================

Extends webshop_product_filter module with two booleans to use in your theming and controller.


Usage
=====
When adding attributes to your categories, you will see two booleans:
is filter and is in description. If you check it as filter you will see it in
advanced search for that category , if you check is in description, you can use
it in your theme or description page.
