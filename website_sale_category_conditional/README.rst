=================================
Website sale category conditional
=================================

* adds published
* hides empty categories automatically
* extends domain to filter only through published categories 
* extends template sale to show in menu published categories
* extends backend view to manage such categories.

these categories are cross-multiwebsite

* works in conjunction with the filtering of the products 
  that are viewed and managed per-website

Usage
=====
From category form, unselect "published" checkbox, to hide that category from website.
